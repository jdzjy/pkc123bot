123bot 使用指南

Bot 常用命令
  • /start - 启动机器人，查看状态
  • /share [关键词] - 搜索文件并创建分享链接
  • /info - 查看123云盘账号信息
  • /add [链接] - 添加监控过滤词
人形命令 (Userbot)
  • -s123 [关键词] - 搜索并生成 JSON 秒传文件
  • -mc - (回复消息) 解析链接或转存 JSON 文件
302 播放服务

将文件路径追加到 URL 后即可获取直链：

  • 基础接口: http://IP:12366/d/文件名.mp4
  • 多线播放: /xiaohao1/ 至 /xiaohao5/
注意事项
  • 请勿在公共场合泄露 Web 管理密码
  • 人形功能需先在 Web 端完成 TG 登录
  • 115转存失败请尝试更新 Cookie 并重启
